####Counting out problem


##### To run the program

```
sbt "run <n> <k>"

``` 
- n is the number of people on the table

- k is the step rate, (a -ve k indicates counting in anti-clockwise direction)  

Example sbt "run 3 2"

output:
```
the position you need to stand at is : 3
```

```
sbt test
```
